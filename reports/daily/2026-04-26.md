## Key Learnings (Top 3)

1. **iOS-Simulator coordinate calibration via `cliclick` + macOS blue-pixel analysis: `macOS_y = 79.4 + iOS_point × 1.078`** — Discovered Sunday 12:00 in pitagents while trying to drive the iOS Simulator headlessly. Standard `cliclick` documentation doesn't account for the iOS-Simulator window's titlebar offset or pixel-density scaling on a portrait 1080×1920 display. The empirical fix: take a macOS screenshot, find a known iOS button by its blue pixel signature, derive the linear formula. Repurposable for *any* future iOS-Simulator automation in a sandbox/CI environment without Xcode UI driving — a class of testing AIBrain has been hitting for two weeks. Source: `sync/journals/projects_pitagents/2026-04-26.md` Session 12:00.

2. **JWT URL-safe base64 silent-decode trap (cross-platform reference)** — Swift's `Data(base64Encoded:)`, Java's standard `Base64`, Python's `b64decode`, and JS `atob()` all reject URL-safe base64 (`-`/`_`) — JWT payload segments use URL-safe base64 by spec, so decoding silently returns nil/empty. The bug only surfaces when a token's bytes happen to include `-` or `_`, so unit tests with a fixed dev token can pass while production fails. Discovered during the Sunday iOS code review (P0 #1). Per-platform fixes documented for Swift/Kotlin/Python/JS in `sync/memory/-Users-joehe-workspace-interview-AIBrain/reference_jwt_url_safe_base64.md`. Symptom: profile/account fields read from JWT claims (`email`, `role`, `shop_id`) are *blank for some users* — easy to misdiagnose as "backend forgot to set the claim."

3. **"Just do it" autonomy preference now memorialized as cross-project feedback** — Joe explicitly asked Claude in pitagents to stop asking "want me to also create X?" and instead just execute reasonable next steps. Saved as `sync/memory/-Users-joehe-workspace-projects-pitagents/feedback_autonomy.md`. Applies cross-project: skip the confirmation question for any reversible, non-shared-system action; only stop to ask when on the dangerous-ops blacklist or when costs are irreversible. Shifts the default from "confirm before adding" to "add and explain in the diff."
