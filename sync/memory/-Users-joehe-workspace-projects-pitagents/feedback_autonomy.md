Just do it. If something seems like a reasonable next step or a missing piece, create it without asking first.

**Why:** User finds confirmation prompts disruptive — they slow down the flow and add friction for decisions Claude can make independently.

**How to apply:** When about to ask "want me to also create X?" or "should I add Y?" — skip the question and do it. Only stop to ask when the action is irreversible, affects shared systems, or is on the dangerous operations blacklist.
