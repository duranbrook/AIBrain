name: Don't pause between phases
description: Keep executing through multi-phase plans without asking permission to continue
type: feedback

Don't stop and ask "want me to continue with Phase X?" between implementation phases. The default is relentless work — keep going through all phases unless explicitly told to stop.

**Why:** User has relentless mode as default in CLAUDE.md. Pausing between phases wastes time and contradicts the configured working mode.

**How to apply:** After completing a phase, immediately start brainstorming/planning the next phase. Only pause if blocked or if there's a genuine decision point that changes the spec.
